# 舱外感知 Demo 快速上手指南

> 本指南面向**第一次运行**本项目的测试人员，无需阅读任何源代码。

---

## 1. 环境准备（忽略，使用已有环境）

确保电脑已安装 Python 3.8+，然后在项目根目录打开终端，运行：

```bash
pip install -r requirements.txt
```

验证环境是否正常：

```bash
python test-environment.py
```

看到 `所有环境配置完美` 即可。如果 YOLOv8 导入失败也没关系——模拟模式不需要它。

---

## 2. 三种运行模式

### 模式 A：模拟模式（推荐先跑这个）

不需要摄像头，不需要模型文件，开箱即用：

```bash
python demo_external.py --mode sim
```

程序会自动循环演示 5 个驾驶场景（共 22 秒一轮）：

| 时间段 | 场景 | 预期表现 |
|--------|------|---------|
| 0~5s | 安全跟车 (25m) | 绿色框，ALL CLEAR |
| 5~10s | 前车减速 (25→10m) | 黄色框，CAUTION |
| 10~14s | 急速接近 (10→2m) | 红色框闪烁 + 蜂鸣声 |
| 14~18s | 前车加速离开 | 恢复绿色 |
| 18~22s | 右侧摩托车出现 | 黄色框 |

### 模式 B：视频文件测试

```bash
python demo_external.py --mode video --source 你的视频路径.mp4
```

需要项目根目录下有 `yolov8n.pt` 模型文件。

### 模式 C：实时摄像头

```bash
python demo_external.py --mode camera --source 0
```

`0` 是默认摄像头编号，如果有多个摄像头可以改成 `1`、`2`。

## 3. 快捷键

| 按键 | 功能 |
|------|------|
| `Q` / `ESC` | 退出 |
| `S` | 截图保存到当前目录 |
| `P` | 暂停 / 继续 |
| `+` / `-` | 调高 / 调低检测置信度 |

---

## 4. 画面说明

- **顶部状态栏**：左侧显示 FPS 和检测到的车辆数量，右侧显示全局风险状态
- **检测框颜色**：绿色 = 安全，黄色 = 注意，红色 = 碰撞警告
- **框上方标签**：车辆类别、距离（米）、TTC（碰撞时间，秒）、相对速度
- **声音警报**：连续 2 帧以上出现红色警告时，会发出蜂鸣音（3 秒冷却，不会连续轰炸）
